# 🏥 TurnoPlus

Una aplicación móvil desarrollada en **React Native (Expo)** y **TypeScript** diseñada para el personal de enfermería en Colombia. Permite proyectar ingresos mensuales de manera precisa calculando automáticamente los recargos nocturnos y dominicales según la legislación laboral vigente (Ley 2026).

## ✨ Características Principales
* **Cálculo Financiero Dinámico:** Motor de cálculo que procesa recargos nocturnos (35%) y festivos (75%) cruzando la medianoche.
* **Persistencia Local:** Gestión de estado y almacenamiento local seguro mediante `AsyncStorage`.
* **Tipado Estricto:** Código 100% tipado con TypeScript para garantizar escalabilidad y cero errores en tiempo de ejecución.
* **Diseño UI/UX:** Interfaz fluida, manejo de SafeArea y modales nativos.

## 🛠️ Tecnologías
* React Native / Expo
* TypeScript
* React Native Calendars
* AsyncStorage
